# Data Science Projects

This repository contains multiple machine learning and deep learning projects:

## Projects Included:
- NLP Sentiment Analysis using TF-IDF + Logistic Regression
- Time Series Forecasting using RNN, LSTM, GRU
- Customer Segmentation using KMeans & DBSCAN
- ANN classification on Sonar dataset
- Recommendation System using Cosine Similarity

## Tools Used:
Python, Pandas, Scikit-learn, TensorFlow, Keras, Matplotlib, Seaborn
